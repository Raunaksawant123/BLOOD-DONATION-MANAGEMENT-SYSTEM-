# Blood Donation Management System

## 📌 Project Description
A web-based system to manage blood donors, recipients, and blood availability.

## 🚀 Features
- Donor registration
- Blood request system
- Admin dashboard
- Database management

## 🛠 Tech Stack
- Frontend: HTML, CSS, JS
- Backend: Python / PHP / Node
- Database: MySQL

## ⚙️ Setup Instructions
1. Import database from database/blood_db.sql
2. Run backend server
3. Open frontend
